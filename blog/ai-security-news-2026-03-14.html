🔥 今日头条：LLM 安全护栏可被绕过 - 攻击成功率 99%

来源: Palo Alto Unit 42 | Infosecurity Magazine

Palo Alto Networks 的 Unit 42 研究实验室发现，用于防止恶意使用的 LLM 安全护栏本身可以被提示注入攻击绕过。研究人员开发的 AdvJudge-Zero 自动化模糊测试工具，攻击成功率达到 99%。

关键发现

  • LLM "AI Judges"（用于执行安全策略的模型）可被操纵授权违规操作
  • AdvJudge-Zero 通过分析模型的下一个令牌概率分布识别漏洞
  • 攻击使用低困惑度令牌（如 markdown 符号、列表标记）绕过检测
  • 甚至最大的 700 亿参数模型也易受攻击
  • 解决方案：通过对抗性训练可将攻击成功率从 99% 降至接近零

影响范围：所有使用 LLM 作为安全护栏的企业 AI 系统

⭐ OpenAI 收购 Promptfoo 加强 AI 代理安全测试

来源: Infosecurity Magazine | 2026-03-10

OpenAI 收购 AI 代理安全测试公司 Promptfoo，以加强企业 AI 生态系统的安全框架。Promptfoo 提供 LLM 和 AI 代理漏洞扫描、红队测试、模型评估等工具。

  • 被 25% 的财富 500 强公司使用

n8n 零点击漏洞 - 无需认证即可完全控制服务器

来源: Pillar Security | 2026-03-12

Pillar Security 研究人员在 n8n 工作流自动化平台发现两个新的关键漏洞，其中一个可零点击、无需认证即可完全接管服务器。

CVE 编号 CVSS 评分 描述 影响
CVE-2026-27493 9.5 关键 零点击、无需认证 公开表单字段可执行任意 shell 命令
CVE-2026-27577 9.4 关键 沙箱逃逸 解密所有存储的凭据

修复建议：更新到 n8n 版本 2.10.1、2.9.3 或 1.123.22，并轮换所有存储的凭据。

CISA 紧急指令：Cisco SD-WAN 漏洞被积极利用

来源: CISA | 2026-03-12

美国网络安全和基础设施安全局（CISA）发布紧急指令，警告攻击者正在积极利用 Cisco Catalyst SD-WAN 基础设施漏洞。

  • CVE-2026-20127：CVSS 10（最高严重级别）
  • 认证绕过漏洞，允许未经认证的攻击者获得管理员访问权限
  • 可操控网络配置或中断政府系统流量
  • 联邦机构需在 3 月 23 日前完成修复
